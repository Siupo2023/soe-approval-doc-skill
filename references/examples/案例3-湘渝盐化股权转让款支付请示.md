# 案例3：湘渝盐化股权转让款支付请示

**类型**：付款类/股权转让类
**日期**：2021年3月9日
**关键特点**：
- 涉及重大资产重组背景
- 股权转让款计算清晰
- 说明提前支付原因（节约财务费用）
- 包含详细的价格计算公式

---

# 关于支付工银投资持有湘渝盐化股权转让款的请示

**公司领导：**

2021年以来，重庆湘渝盐化股份有限公司（以下简称"湘渝盐化"）资产证券化工作（以下简称"本次交易"）加速推进，上市公司雪天盐业集团股份有限公司（以下简称"雪天盐业"）已就本次交易事宜公告重组预案。

为保障本次交易的顺利实施，经集团党委会、董事会研究决定受让工银金融资产投资有限公司（以下简称"工银投资"）持有湘渝盐化全部股权，并于2021年2月22日签订《股权转让协议》。根据有关约定及资金筹措情况，拟近期支付股权转让款。

根据集团公司与工银投资的《股权转让协议》的约定，集团公司应于2021年3月31日前一次性支付全部股权受让款，受让价格=【本金】60,000万元+【未实现收益】（本金60,000万元*5.8%*工银投资出资日（含）至转让价款支付日（不含）之间的天数/365）/0.75。

为节约财务费用，经与工银投资协商，拟于2021年3月10日支付全部款项。经双方测算一致，截至2021年3月10日，按照资金实际占用天数438天计算，股权受让款总金额为人民币65,568.00万元整。

现提请支付该笔款项，特此请示！

---

**落款**：2021年3月9日

---

## 分析要点

### 写作优点

1. **背景完整**：
   - 说明资产证券化工作推进情况
   - 说明党委会、董事会决策依据
   - 引用《股权转让协议》具体约定

2. **计算公式清晰**：
   ```
   受让价格 = 本金(60,000万元) + 未实现收益
   未实现收益 = (本金 × 5.8% × 天数 ÷ 365) ÷ 0.75
   ```
   - 公式透明，便于审计
   - 参数明确，便于复核

3. **提前支付理由充分**：
   - "为节约财务费用"
   - "经与工银投资协商"
   - 提前21天支付（3月10日 vs 3月31日）

4. **金额精确**：
   - 总金额：65,568.00万元
   - 占用天数：438天
   - 小数点后两位精确

### 适用场景

- 股权转让款支付
- 大额资金支付申请
- 投资款项支付
- 资产收购付款

### 特殊技巧

**计算公式的呈现方式**：
- 使用【】标注关键参数
- 分层展开复杂计算
- 保留计算过程便于审核

**时间节点的表述**：
- 截止时间："应于2021年3月31日前"
- 实际时间："拟于2021年3月10日"
- 提前原因："为节约财务费用"

### 结构特点

```
开篇（背景） → 决策依据 → 协议约定 → 支付计划 → 现提请请示
```

**不设小标题**：整个请示一气呵成，不需要分层小标题，适合逻辑清晰、内容单一的事项。
