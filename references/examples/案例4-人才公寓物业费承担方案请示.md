# 关于人才公寓物业费承担方案的请示

**公司领导：**

根据湘江新区（长沙高新区）党工委（岳麓区委）人才引进政策，纤维院于2025年11月24日与长沙麓谷公共租赁住房管理有限公司签订了《租赁合同》，并于2025年11月18日成功获批申请麓谷人才家苑免租公寓。现将有关事项请示如下：

## 一、基本情况

纤维院依托长沙市人才房政策支持，申请获批麓谷人才家苑1栋免租公寓10套（36间），享受免租金待遇。申请时按所有符合条件员工均申请的原则进行统一申报，最终实际租赁并投入使用10套（36间）房屋。

截至目前，36间房屋使用情况为：15间由公司员工入住（魏梦西等11人），总面积529.49平方米，16间预留用于复旦大学"可穿戴电子织物的工程化研究"校企联合培养项目师生使用，共计753.47平方米。

根据《租赁合同》约定，物业费单价为1.96元/㎡/月，每月合计2514.60元，按季度缴纳。租赁期已满一个季度，现已收到物业公司缴费通知，需缴纳第一季度物业管理费，共计7543.8元。

## 二、物业费分摊方案

建议由实际入住员工按其入住房屋对应面积承担相应物业管理费，对于预留用于复旦大学联合培养项目的21间房屋，随着项目推进，师生入住情况可能发生动态变化，为便于统筹管理，该部分房屋对应的物业管理费建议由公司统一承担。

因此第一季度物业费总额7543.8元，其中员工个人承担3113.4元（529.49平方米×1.96元×3个月），公司统一承担4430.40元。公司承担的物业费建议从管理费用或其他相应经费中列支。

## 三、请示结果

请审批上述人才公寓物业费分摊方案，同意员工实际入住房屋对应物业费由个人承担，预留复旦联合培养项目房屋对应物业费由公司统一承担并从管理费用或相应经费中列支。

妥否，请批示。

**附件：**
1. 《人才公寓租赁合同》
2. 《租赁房屋清单、物业费明细》

## 分析要点

### 写作优点

1. **政策依据明确**：
   - "根据湘江新区（长沙高新区）党工委（岳麓区委）人才引进政策"
   - 说明政策背景和支持力度

2. **数据完整精确**：
   - 房屋数量：10套（36间）
   - 员工入住：15间，529.49平方米
   - 预留用途：16间，753.47平方米
   - 物业费单价：1.96元/㎡/月
   - 总金额：7543.8元

3. **分摊方案合理**：
   - 员工承担：实际使用部分（529.49㎡ × 1.96 × 3 = 3113.4元）
   - 公司承担：预留项目部分（4430.4元）
   - 理由充分："便于统筹管理"、"动态变化"

4. **计算过程透明**：
   ```
   员工承担 = 529.49 × 1.96 × 3 = 3113.4元
   公司承担 = 7543.8 - 3113.4 = 4430.4元
   ```

5. **附件支撑**：
   - 《人才公寓租赁合同》
   - 《租赁房屋清单、物业费明细》
