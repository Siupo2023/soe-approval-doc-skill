# 关于采购法律顾问服务的请示

**公司领导：**

根据公司经营发展需要，为确保各项业务合法合规开展，特别是技术开发、成果转化等核心业务提供法律保障，拟采购法律顾问服务。现将有关事项请示如下：

## 一、基本情况

党群综合部于2025年11月6日向多家律师事务所发出法律顾问服务询价函，服务期限为合同签署后3个月。该项目已获得预算批准，预算金额为人民币3万元。为公司提供常年法律顾问服务，包括日常法律咨询、合同审查、法律风险防控、协助建章立制、参与重要事项法律论证等相关法律服务。

## 二、询比价结果

此次询价共收到4家律师事务所的报价函：

1. 北京德恒律师事务所（汤敏志团队）：2.5万元
2. 湘军麓和律师事务所（李含英团队）：2.5万元
3. 湖南方哲律师事务所（夏德仁团队）：2.8万元
4. 湖南仁本律师事务所（唐溪团队）：3万元

## 三、拟选结果及原因

建议选定湘军麓和律师事务所作为我司常年法律顾问服务供应商，该所报价2.5万元，为本次询价最低价，且作为本地所，能够及时提供现场法律服务。且此次服务团队为该所主任亲自带队，服务质量有一定保证。

妥否，请批示。

**附件：**
1. 法律顾问服务询价函
2. 各律师事务所报价函

## 分析要点

### 写作优点

1. **开篇明确需求**：
   - "根据公司经营发展需要"
   - 特别说明核心业务保障需求（技术开发、成果转化）

2. **基本情况完整**：
   - 时间：2025年11月6日发出询价函
   - 服务期限：3个月
   - 预算：已获批，3万元
   - 服务内容：详细列出5项服务

3. **询比价透明**：
   - 明确数量：共4家报价
   - 清晰列表：律所名称+团队+价格
   - 便于对比决策

4. **选择理由充分**：
   - 价格因素："为本次询价最低价"
   - 服务因素："本地所，能够及时提供现场法律服务"
   - 团队因素："主任亲自带队，服务质量有保证"

5. **附件齐全**：
   - 询价函
   - 各家报价函
   - 支撑决策依据

### 可借鉴的表达

- "根据......需要，为确保......，特别是......提供......保障"
- "......于......向......发出......询价函"
- "该项目已获得预算批准，预算金额为......"
- "包括......、......、......等相关......服务"
- "此次询价共收到......的报价函"
- "建议选定......作为我司......供应商"
- "该所报价......，为......最低价"
- "且作为......，能够......"
- "且此次......为......，......有一定保证"

### 结构特点

```
开篇（需求） → 基本情况（时间、预算、内容） → 询比价结果（4家对比） → 拟选结果及原因 → 结尾
```

### 数据呈现技巧

对于多家对比的报价，使用编号列表更清晰：

```
1. 供应商A（团队）：价格
2. 供应商B（团队）：价格
3. 供应商C（团队）：价格
4. 供应商D（团队）：价格
```

### 论证逻辑

选择理由的三个层次：
1. **价格**：最低价（经济效益）
2. **服务能力**：本地所，响应快（服务保障）
3. **团队实力**：主任带队（质量保证）

这种多维度论证方式使请示更有说服力。
