# 国企公文格式规范详解

**版本**：v1.0
**适用范围**：国有企业内部请示、呈批件等公文
**更新日期**：2026-03-25

---

## 一、内容质量要求

### 1.1 六大质量标准

1. **文种正确，文体相符**
   - 准确使用"请示"、"报告"、"通知"等文种
   - 文体与内容、目的相匹配

2. **主体鲜明，文题相符**
   - 标题准确概括主要内容和文种
   - 主体明确，避免歧义

3. **结构严谨，层次分明**
   - 逻辑清晰，条理清楚
   - 层次递进，环环相扣

4. **短小精悍，通俗易懂**
   - 篇幅简短，避免冗长
   - 语言简洁，易于理解

5. **逻辑缜密，用词准确**
   - 推理严密，论证充分
   - 词语精准，表达恰当

6. **一文一事，简单明了**
   - 一份公文只讨论一个事项
   - 请示事项明确具体

### 1.2 语言风格要求

- ✅ **简明扼要，层次分明，突出重点**
- ✅ **语言文字规范准确、严肃严谨，避免口语化**
- ✅ **内容简洁明快，篇幅力求简短**
- ✅ **用语规范，语言文字使用力求准确、规范**

### 1.3 格式要求

- 格式要正确
- 结构要完整
- 排版要规范

## 二、页面格式规范

### 2.1 纸张与页边距

```
纸张规格：A4，纵向
页边距设置：
  - 上边距：25 毫米
  - 下边距：25 毫米
  - 左边距：28 毫米
  - 右边距：26 毫米
```

### 2.2 标题格式

```
字体：方正小标宋简体
字号：二号字
行距：固定值 30 磅
对齐：居中排布
```

**标题要求**：
- 准确简要地概括主要内容和文种
- 除法规、规章名称加书名号外，一般不使用标点符号
- 标题有回行时，要做到词意完整，排列对称

### 2.3 正文格式

```
字体：仿宋GB2312
字号：三号字
行距：固定值 28 磅
段落缩进：首行左边空 2 个字
回行：顶格
```

### 2.4 层次结构规范

**四级标题顺序**（不能逆用）：

| 层级 | 编号格式 | 示例 |
|------|---------|------|
| 第一层 | 汉字 + 顿号 | 一、二、三、 |
| 第二层 | 括号 + 汉字 | （一）（二）（三） |
| 第三层 | 阿拉伯数字 + 点 | 1. 2. 3. |
| 第四层 | 括号 + 阿拉伯数字 | （1）（2）（3） |

**使用示例**：
```
一、基本情况
（一）背景
1. 具体情况
（1）细节说明
```

### 2.5 附件格式

**位置**：正文下空 1 行，左空 2 字

**格式**：
```
  附件：1. XXXXX说明
        2. XXXXX清单
```

**要求**：
- 标识"附件"二字，后标全角冒号和附件名称
- 附件如有序号，使用阿拉伯数字（序号与名称间用"."隔开）
- 附件名称后不加标点符号
- 附件名称如有回行，回行后的首字应当与附件名称首字对齐
- 附件说明中的序号和名称必须与附件的序号名称一致

### 2.6 落款格式

**组成**：单位、部门成文日期和印章

```
[部门名称]
YYYY年MM月DD日
（印章盖在日期上）
```

**要求**：
- 成文日期以负责人签发的日期为准
- 使用汉字标注，标全年、月、日
- "零"写为"0"
- 成文日期上加盖印章

## 三、Word 文档格式实现

### 3.1 字体映射

由于系统可能不包含所有字体，使用以下备选方案：

| 规范字体 | 备选字体 | 说明 |
|---------|---------|------|
| 方正小标宋简体 | 方正小标宋、华文中宋、黑体 | 标题用 |
| 仿宋GB2312 | 仿宋、仿宋_GB2312 | 正文用 |

### 3.2 页面设置参数（单位：磅）

```
1 毫米 ≈ 2.835 磅

上边距：25mm × 2.835 ≈ 70.9 磅
下边距：25mm × 2.835 ≈ 70.9 磅
左边距：28mm × 2.835 ≈ 79.4 磅
右边距：26mm × 2.835 ≈ 73.7 磅
```

### 3.3 行距设置

```
标题行距：固定值 30 磅
正文行距：固定值 28 磅
```

### 3.4 字号对照

```
二号字 = 22 磅
三号字 = 16 磅
```

## 四、常见问题与注意事项

### 4.1 格式问题

1. **页边距设置不准确**
   - 必须严格按照上25、下25、左28、右26（毫米）设置

2. **字体不匹配**
   - 标题必须使用方正小标宋简体（或备选字体）
   - 正文必须使用仿宋GB2312（或备选字体）

3. **行距不规范**
   - 标题：固定值 30 磅
   - 正文：固定值 28 磅

4. **层次结构混乱**
   - 严格按照"一、→（一）→1.→（1）"的顺序
   - 不可逆用或跳级使用

### 4.2 内容问题

1. **语言口语化**
   - 避免使用"咱们"、"挺"、"特别"等口语词汇
   - 使用规范的书面语表达

2. **层次不分明**
   - 每个部分应当有明确的小标题
   - 内容按逻辑顺序排列

3. **篇幅过长**
   - 简明扼要，突出重点
   - 避免冗余表述
